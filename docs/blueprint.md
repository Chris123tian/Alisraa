# **App Name**: Alisraa International

## Core Features:

- Homepage Replication: Replicate the homepage layout, content, and interactive elements from the provided website, ensuring a visually accurate representation.
- About Us Section: Mirror the 'About Us' section, accurately presenting the company's mission, values, and team as depicted on the original website.
- Services Showcase: Duplicate the services showcase, including descriptions, images, and any unique features used to highlight the company's offerings.
- Contact Form: Recreate the contact form, enabling users to submit inquiries, with form submissions forwarded to a specified email address.
- Blog Section: Implement a blog section with a content management system (CMS) to update and display posts, replicating the look and feel of the original blog.

## Style Guidelines:

- Primary color: Deep teal (#008080), reflecting stability and trust. 
- Background color: Soft light grayish-blue (#E0F0F0), for a clean, professional feel.
- Accent color: Light cyan (#70DBD4), used to draw attention to key interactive elements such as CTAs.
- Body and headline font: 'Inter', a grotesque sans-serif, chosen for its neutral, modern feel, and suitability for both headlines and body text.
- Employ a set of clean, professional icons to visually represent key service offerings.
- Adopt a modern, responsive layout, ensuring the website is fully accessible across all devices.
- Integrate subtle transitions and animations to enhance user experience without detracting from the site's professional appearance.
